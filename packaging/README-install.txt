puyo2 release archive

Included binaries:
- dfield
- dpuyo
- nazo
- pnconv
- pnsolve
- pnsolve2simus

These archives do not bundle sprite assets.

For image-oriented commands, set PUYO2_CONFIG to a directory containing:
- puyos.png
- puyos_transparent.png
- puyos_shape.png

You can also place those files under an images/ directory in your working tree.

Source and install documentation:
https://github.com/wata-gh/puyo2
